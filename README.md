# BEBOU

### Better Enhancement and Better Optimization Upgrade

A curated modpack built for a cozy SMP experience with friends. This was originally made for a private SMP, but feel free to use it !  

Fully **vanilla compatible** — use the client pack, server pack, or both independently.

## What's Inside

- ⚡ **Optimization**
- 🧭 **Better UI/UX**
- 🍃 **QoL** : Voxy, Inventory Sorting, Light Overlay, Schematics & Gravestone on Death
- 🌄 **Shaders** : Comes pre-packaged with some shaders, most of which are compatible with Voxy
- 🌧️ **Atmosphere** : Cozier rain sounds
- 🎮 **Controller Support**
- 💬 **Multiplayer** : Better chat, player locators & BlueMap live web map

## Optional Features

By default, BEBOU comes with some features toggled off. Here is how to enable each of them :

### Voxy

To enable Voxy, go to `Options > Video Settings` and toggle `Voxy > General > Enable Voxy`
